# Seedance 2.0 API Parameter Reference

## Model IDs

| Model | Model ID | Features |
|-------|----------|----------|
| Seedance 2.0 | `doubao-seedance-2-0-260128` | Highest quality |
| Seedance 2.0 Fast | `doubao-seedance-2-0-fast-260128` | Faster, lower cost |

## Create Task Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `model` | string | Yes | Model ID |
| `content` | array | Yes | Input content array, see below |
| `ratio` | string | No | Output aspect ratio, default `16:9` |
| `duration` | int | No | Duration in seconds, range 4~15, default 5 |
| `generate_audio` | bool | No | Whether to generate video with audio |
| `watermark` | bool | No | Whether to add a watermark |
| `tools` | array | No | Tool list, e.g. `[{"type": "web_search"}]` |
| `service_tier` | string | No | `"flex"` for offline inference |
| `return_last_frame` | string | No | `"true"` to also return the last frame image |

## Content Array Element Formats

### Text (must be first)

```json
{
  "type": "text",
  "text": "Prompt content"
}
```

### Reference Images

```json
{
  "type": "image_url",
  "image_url": {"url": "https://..."},
  "role": "reference_image"
}
```

Image role options:
- `reference_image`: Multimodal reference image
- `first_frame`: First frame image-to-video
- `last_frame`: Last frame image-to-video

### Reference Videos

```json
{
  "type": "video_url",
  "video_url": {"url": "https://..."},
  "role": "reference_video"
}
```

### Reference Audio

```json
{
  "type": "audio_url",
  "audio_url": {"url": "https://..."},
  "role": "reference_audio"
}
```

### Using Platform Assets (Virtual Avatars / Authorized Real Persons)

URL format: `asset://<asset_id>`

```json
{
  "type": "image_url",
  "image_url": {"url": "asset://asset-20260401123823-6d4x2"},
  "role": "reference_image"
}
```

## Input Limits

| Modality | Quantity Limit |
|----------|----------------|
| Text | Required, 1 item |
| Images | 0~9 |
| Videos | 0~3 |
| Audio | 0~3 |

**Unsupported combinations:**
- Text + Audio (without images/videos)
- Audio-only input

## Output Specifications

### Resolution

| Model | Available Resolutions |
|-------|-----------------------|
| Seedance 2.0 | 480p, 720p, 1080p |
| Seedance 2.0 Fast | 480p, 720p |

Default: 720p (set via the `resolution` parameter, e.g. `"resolution": "720p"`)

### Aspect Ratio

| Value | Description |
|-------|-------------|
| `16:9` | Landscape (default) |
| `9:16` | Portrait |
| `1:1` | Square |
| `4:3` | Traditional landscape |
| `3:4` | Traditional portrait |
| `21:9` | Ultrawide |
| `adaptive` | Follows the input image aspect ratio |

### Duration

- Range: 4~15 seconds
- Default: 5 seconds

## Prompt Asset Referencing Rules

Use the "asset type + sequence number" format in prompts to reference assets:
- `Image 1`, `Image 2`... → The Nth asset with `type=image_url` in the content array (counting from 1)
- `Video 1`, `Video 2`... → The Nth asset with `type=video_url` in the content array
- `Audio 1`, `Audio 2`... → The Nth asset with `type=audio_url` in the content array

**Note**: You cannot reference assets directly using Asset IDs in the prompt.

## Face Policy

| Output Type | Validity Period |
|-------------|-----------------|
| Videos containing faces generated by Seedance 2.0/Fast | 30 days |
| Last frame images from videos generated by Seedance 2.0/Fast | 30 days |
| Images containing faces generated by Seedream 5.0 lite text-to-image | 30 days |

Conditions: Same account, original output (not re-edited), within the validity period.

### Authorized Real-Person Assets

After completing real-person verification and authorization, use via `asset://<asset_id>`.

## FAQ

### Video Frame Jumps

In first/last frame scenarios, if the input image aspect ratio does not match the output video aspect ratio, frame jumps may occur.

**Solutions**:
1. Set `ratio` to `adaptive`
2. Or crop the input image to the pixel dimensions corresponding to the target aspect ratio

### Video URL Expiration

Generated video URLs are valid for only 24 hours. Recommendations:
- Download and save promptly
- Or use the Volcengine TOS data subscription feature for automatic transfer
